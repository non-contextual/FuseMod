# FuseMod

> 其他语言：[English](./README.md) · 简体中文
>
> 把《塞尔达：王国之泪》的「融合」机制搬到 Minecraft 1.21.11。

把材料融合到武器、工具、盾牌上，获得真正的战斗与挖掘加成。任何时候都能解融，把材料拿回来。

---

## 怎么用

**融合**：副手拿可融合的材料，主手拿支持融合的武器/工具/盾，然后 **潜行 + 右键**。

**解融**：主手拿融合后的物品（副手可以空），再次 **潜行 + 右键**。材料会还到背包；背包满了就掉在脚下。

每件装备同一时间只能融合一种材料。再融合会替换原有的融合，想换的话先解融。

---

## 支持的装备

| 装备类型 | 效果 |
|---|---|
| 剑、斧、三叉戟 | 武器 `+X 攻击伤害` |
| 镐、锹、锄 | `+X.X 挖掘效率` |
| 盾牌 | 视觉标签（完整效果在 v1.2.0 加入） |

效果叠加在物品原有数值之上，包括附魔和其他修饰。解融时完全清除。

## 箭矢融合

把材料直接融合到箭上，可以从任何弓或弩发射特殊弹道。

**做一支融合箭**：副手拿可融合的材料，主手拿一支普通箭，然后 **潜行 + 右键**。消耗 1 支箭 + 1 个材料。

**飞行中触发**：拉弓或弩时副手拿可融合的材料，融合箭会自动发射，每次消耗一个材料。

融合箭是普通箭物品，可以堆叠到 64，发射后能捡回来。

### 箭矢效果

| 材料 | 效果 |
|---|---|
| 末影珍珠 | **传送**：射手传送到落点 |
| 避雷针 | **闪电**：命中处召唤闪电；下雨时连击附近生物 |
| 回响碎片 | **追踪**：自动转向最近的敌对生物（半径 12） |
| 海晶碎片 | **散射**：命中后分裂成 3 支箭 |
| 线 | **蛛网**：在落点放置一块蛛网（15 秒后消失） |
| 黏液球 | **弹跳**：碰墙反弹，最多 3 次 |
| 下界之星 | **新星**：非破坏性爆炸 + 给附近生物施加凋零 I |
| 凋灵骷髅头骨 | **凋灵**：对目标施加凋零 II（5 秒） |

没有箭矢效果的材料还是会带颜色染色，融合到近战武器时同样有攻击/火/击退加成。

## Tooltip

融合后的物品会显示一行金色 **Fused: `<material>` [bonus]**，加一句解融提示。

未融合但兼容的物品会有暗灰色提示 **"副手拿材料，潜行 + 右键融合"**。

## 从源码构建

```bash
git clone https://github.com/non-contextual/FuseMod
cd FuseMod
./gradlew build
```

输出 JAR 在 `build/libs/fusemod-1.2.0.jar`。
